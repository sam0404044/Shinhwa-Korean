# 前端素材資料夾結構

```
assets/
├─ images/
│  ├─ hero/
│  │  ├─ hero-kdog-landing-desktop.png   # 16:9 首屏主視覺（滿版）
│  │  └─ hero-kdog-landing-mobile.png    # 4:5 / 9:16 手機主視覺（滿版）
│  ├─ courses/
│  │  ├─ beginner.png                    # 入門（構圖：狗+學習道具）
│  │  ├─ grammar.png                     # 文法（狗+文法/句型視覺）
│  │  ├─ speaking.png                    # 會話（狗+對話泡泡圖形但不放字）
│  │  └─ exam.png                        # 檢定（狗+勾選/考試感道具）
│  ├─ content/
│  │  ├─ news.png                        # 最新消息/文章入口用插畫
│  │  └─ article.png                     # 文章內頁首圖（可選）
│  ├─ testimonials/
│  │  ├─ avatar-01.png                   # 方形滿版、留安全邊界可裁圓
│  │  ├─ avatar-02.png
│  │  └─ avatar-03.png
│  └─ ui/
│     ├─ banner-cta.png                  #（可選）情緒橫幅插畫，無字
│     └─ empty-state.png                 #（可選）無資料狀態插畫
└─ decor/
   ├─ sticker-heart.png                  # 愛心貼紙風（無字）
   ├─ sticker-star.png
   ├─ sticker-sparkle.png
   └─ sticker-line.png                   # 手繪線條點綴
```

風格參考：柔和色、扁平插畫、白色卡通狗角色、韓語學習平台（kolanee 風）。

**使用方式**：將對應圖片放入上述路徑即可，首頁已依此路徑引用。若圖片尚未就緒，Hero 會顯示淺綠背景，課程卡／最新消息／學員頭像會顯示柔和色塊佔位。
